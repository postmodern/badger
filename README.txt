Badger! Badger! Badger!
=======================

Badger is a fear-less, drop-in and miminal systems management solution.

Features
--------

* Compiles a portable statically linked executable.
* Compiles a `libbadger-static` library for developers.
* Uses ZeroMQ for networking and message delivery.
* Uses MessagePack for the message format.

Requirements
------------

* libzmq (http://www.zeromq.org/)
* libmsgpack >= 0.5.0 (http://msgpack.sourceforge.net)
