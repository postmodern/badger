Badger is a fear-less, robust and portable remote system management tool.

Features
--------

* Written in Ruby.
* Contains an embedded mruby [1] runtime.
* Simple JSON RPC message format.
* Portable statically linked executable.

Requirements
------------

* libc.a and libm.a
* Bison (required by mruby)
* Ruby
* Rake

Build
-----

    $ rake
    $ ./bin/badger
